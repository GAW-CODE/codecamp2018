About

Code Camp is a yearly training camp held by Glen A Wilson CODE and provides a fun environment in which students can learn the basics of computer science and coding. At Code Camp, students are taught the basics of computers as well as web design. Front end languages such as HTML, CSS, and Javascript, and backend web design such as databasing will be taught to the students. Throughout the camp, students will learn to create their own functional websites, meet real computer scientists, and be able to compete in a team for a prize while training for bigger competitions such as the Congressional App Challenge. By the end of the camp, students will be competent enough to create websites and web applications that could be applicable to their everyday life.
